# 第一次编程作业

## 项目描述
该项目实现了数种求解非线性方程的算法，包括二分法、牛顿法和割线法，并应用于不同函数的根的求解。

## 编译与运行
### Linux/macOS
使用以下命令编译和运行项目：
```bash
make run
```

使用一下命令编译TeX文件：
```bash
make report
```

### Windows
使用以下批处理文件进行编译和运行：
```batch
.\Make.bat run
```
